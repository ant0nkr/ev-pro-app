# Панель приладів (Dashboard)

Dashboard — це екран за замовчуванням при відкритті BYD EV Pro. На ньому відображається телеметрія батареї, потужності, запасу ходу та силової установки автомобіля в реальному часі.

---

## Компонування

Dashboard адаптується до доступної ширини екрана.

| Ширина екрана | Компонування |
|---|---|
| Широкий (головний пристрій) | Два стовпці: ліва панель (шкала SOC, потік потужності, картка запасу ходу) + права панель (чотири підвкладки) |
| Вузький | Один прокручуваний стовпець із панеллю вкладок зверху |

---

## Шкала SOC

Шкала стану заряду — це центральний візуальний елемент лівої панелі.

- **Діапазон**: від 0 до 100%
- **Кольорові зони**:

| Діапазон | Колір |
|---|---|
| 0 -- 20% | Червоний |
| 20 -- 40% | Жовтий |
| 40 -- 100% | Зелений |

Числове значення у відсотках відображається великим шрифтом по центру. Коли зв'язок відсутній, у центрі відображається `--`.

**Позначка стану здоров'я батареї (SoH)** розташована під шкалою. Вона вказує на загальний стан високовольтної батареї.

| Значення SoH | Колір позначки |
|---|---|
| >= 80% | Зелений |
| 60 -- 79% | Жовтий |
| < 60% | Червоний |

> [!NOTE]
> На новому автомобілі показник SoH становить 100%. Це значення поступово змінюється протягом терміну експлуатації батареї.

---

## Індикатор потоку потужності

Індикатор потоку потужності розташований під шкалою SOC. Він показує напрямок та величину потужності батареї за допомогою анімованих шевронів (стрілок).

- **Ліворуч (зелені, пульсуючі)**: Батарея заряджається (потужність надходить).
- **Праворуч**: Батарея розряджається (потужність іде до двигуна).
- **"Idle"**: Відображається, коли значення потужності близьке до нуля.

**Кольорові пороги при розрядці:**

| Потужність | Колір |
|---|---|
| < 20 кВт | Синій |
| 20 -- 50 кВт | Жовтий |
| > 50 кВт | Червоний |

---

## Панель запасу ходу

На панелі запасу ходу відображаються три взаємодоповнюючі оцінки поруч.

| Назва | Опис |
|---|---|
| EV Range | Власна оцінка запасу ходу автомобіля |
| Prognosed Range | Розрахована на основі SOC, ємності батареї та вашої середньої ефективності за останні 20 циклів зарядки |
| Actual Range | Розрахована на основі SOC та ефективності поточної поїздки (використовує Prognosed Range, якщо пройдено менше 2 км) |

Усі три значення вказані в кілометрах. Коли автомобіль вимкнений або зв'язок відсутній, кожне з них показує `--`.

> [!NOTE]
> Prognosed Range стає точнішим із накопиченням циклів зарядки. Якщо записано менше 3 циклів, власна оцінка EV Range автомобіля може бути надійнішою.

---

## Підвкладки

Права панель містить чотири вкладки, кожна з яких групує пов'язані показники.

### Cycle (Цикл)

Показники, накопичені з моменту останньої завершеної зарядної сесії.

| Показник | Одиниця |
|---|---|
| Cycle Distance | км |
| Cycle Energy | кВт·год |
| Cycle Efficiency | кВт·год/100 км |
| Est. Distance | км |
| Cycle Time | г:хх |

**Кольорове кодування ефективності:**

| Ефективність | Колір |
|---|---|
| < 23 кВт·год/100 км | Зелений |
| 23 -- 30 кВт·год/100 км | Жовтий |
| > 30 кВт·год/100 км | Червоний |

### Trip (Поїздка)

Показники поточної сесії водіння (з моменту увімкнення автомобіля).

| Показник | Одиниця |
|---|---|
| Trip Distance | км |
| Trip Energy | кВт·год |
| Avg Speed | км/год |
| Trip Efficiency | кВт·год/100 км |
| Est. Distance | км |
| Trip Time | г:хх |

> [!NOTE]
> Середня швидкість розраховується застосунком на основі загальної відстані поїздки та загального часу. Вона не залежить від вбудованого бортового комп'ютера автомобіля.

### All Time (За весь час)

Сукупна статистика за весь час використання, збережена локально.

| Показник | Одиниця |
|---|---|
| Total Distance | км |
| Total Energy | кВт·год |
| Avg Speed | км/год |
| Efficiency | кВт·год/100 км |
| Est. Distance | км |
| Drive Time | г:хх |

Внизу цієї вкладки є кнопка **Reset All Time** з діалогом підтвердження.

### Battery (Батарея)

Телеметрія високовольтної батареї та силової установки в реальному часі. Ця вкладка завантажує детальні дані на рівні комірок за запитом.

**Електричні параметри:**

| Показник | Одиниця |
|---|---|
| Voltage | В |
| Current | А |
| Power | кВт |

**Температури:**

| Показник | Одиниця |
|---|---|
| Max Battery Temp | C |
| Min Battery Temp | C |
| Avg Temp | C |
| Motor Temp | C |

**Напруга комірок:**

| Показник | Одиниця |
|---|---|
| Max Cell Voltage | В |
| Min Cell Voltage | В |
| Cell Delta | мВ |

**Кольорове кодування Cell Delta:**

| Delta | Колір |
|---|---|
| <= 30 мВ | Зелений |
| 31 -- 80 мВ | Жовтий |
| > 80 мВ | Червоний |

> [!NOTE]
> Будь-яке значення, що показує `--`, означає, що датчик недоступний, автомобіль вимкнений або показання некоректне. На деяких моделях температура двигуна може показувати `--` одразу після увімкнення автомобіля.
