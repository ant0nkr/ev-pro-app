# BYD EV Pro — Free / TG-HA / Lite / Pro

Екосистема BYD EV Pro доступна у чотирьох рівнях:

- **Free** — лише застосунок на головному пристрої BYD DiLink. Жодних хмарних функцій. Усі дані залишаються в авто.
- **TG/HA** — Free + інтеграції з Telegram-ботом та Home Assistant (HACS). Дозволяє отримувати сповіщення та керувати авто з Telegram або з HA.
- **Lite** — TG/HA + застосунок-компаньйон на телефоні (Android та iOS). Повноцінне віддалене керування з телефона.
- **Pro** — Lite + апаратний модуль T-Box. Цілодобовий зв'язок з авто, GPS, безключовий доступ та віддалений запуск.

> [!NOTE]
> У версіях **Free** та **TG/HA** усі функції залишаються в авто або проходять через головний пристрій. У версіях **Lite** та **Pro** доступний застосунок-компаньйон. У **Lite** для віддаленого зв'язку використовується Wi-Fi головного пристрою — для цілодобового моніторингу рекомендується встановити **USB Wi-Fi-донгл** в авто (наприклад, з'єднаний з портативним 4G-роутером або точкою доступу телефона). У **Pro** модуль T-Box має власний завжди-увімкнений канал, тож Wi-Fi-донгл не потрібен.

## Безпека

Усі рівні використовують однакові високі стандарти безпеки. Жодних компромісів.

- **AES-256-GCM шифрування** усіх даних та команд між пристроями.
- **HMAC-SHA256** автентифікація кожного повідомлення.
- **Біометричне підтвердження** чутливих команд у застосунку-компаньйоні (Face ID / Touch ID / Android Biometric) — *Lite, Pro*.
- **QR-pairing** з одноразовим токеном — секрет ніколи не передається у відкритому вигляді — *Lite, Pro*.
- **Окремі ключі для кожного авто та кожного телефону.** Компрометація одного пристрою не зачіпає інші.
- **RTT anti-relay захист** для безключового доступу — миттєві ping-pong виклики унеможливлюють атаки, коли зловмисник підсилює BLE-сигнал між телефоном та авто — *Pro*.
- **Локальне сховище секретів** в Android Keystore та iOS Keychain.
- **Жодних серверних паролів** — усі ключі генеруються на пристроях користувача.

**Примітки:**

- **Так\*** означає, що функція потребує активного Wi-Fi-з'єднання головного пристрою. У версії **Lite** для цілодобового зв'язку рекомендується встановити **USB Wi-Fi-донгл** в авто (наприклад, з'єднаний з портативним 4G-роутером або точкою доступу телефона). У версії **Pro** модуль T-Box забезпечує власний канал зв'язку, тож донгл не потрібен.
- **TG** означає, що функція доступна через Telegram-бот. Користувач взаємодіє з ботом у Telegram, а не через окремий застосунок.
- Вбудований 4G-модем DiLink наразі не функціонує надійно — для віддаленого зв'язку у Lite потрібен Wi-Fi.
- Усі шифрування, біометричні перевірки та анти-relay механізми працюють з однаково високим рівнем безпеки в усіх відповідних рівнях.

У версії **Pro** модуль T-Box додає:
- власний незалежний канал зв'язку (Wi-Fi-донгл не потрібен),
- окремий GPS-приймач для точного позиціонування,
- безключовий доступ із захистом від relay-атак (RTT),
- надійні push-сповіщення 24/7,
- віддалений запуск авто.
