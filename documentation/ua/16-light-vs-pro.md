# BYD EV Pro — Light vs Pro

Екосистема BYD EV Pro доступна у двох конфігураціях:

- **Light** — лише застосунок на головному пристрої BYD DiLink + застосунок-компаньйон на телефоні. Без додаткового обладнання.
- **Pro** — Light + апаратний модуль T-Box, встановлений в авто. Модуль додає завжди-увімкнений віддалений канал зв'язку, GPS-трекінг, безключовий доступ за наближенням телефона та надійні push-сповіщення навіть коли авто заглушене.

> [!NOTE]
> У версії Light дистанційні функції працюють лише поки головний пристрій авто має стабільний інтернет. Вбудований 4G-модем DiLink наразі не працює надійно, тому єдиний робочий канал — це Wi-Fi. Якщо ви хочете цілодобовий зв'язок із авто без T-Box, рекомендується встановити **USB Wi-Fi-донгл** (наприклад, підключений до USB-роздачі телефона або портативного 4G-роутера) — головний пристрій залишається в мережі навіть коли авто заглушено. У версії Pro модуль T-Box забезпечує власний незалежний канал зв'язку, тож Wi-Fi-донгл не потрібен.

## Безпека

Обидві версії використовують однакові високі стандарти безпеки. Жодних компромісів через відсутність T-Box.

- **Наскрізне шифрування AES-256-GCM** усіх команд та телеметрії між телефоном, авто та хмарою.
- **HMAC-SHA256** автентифікація кожного повідомлення.
- **Біометричний захист** чутливих команд: відкриття дверей, багажника та віддаленого запуску. Підтверджується Face ID / Touch ID / Android Biometric.
- **QR-pairing** з одноразовим токеном — секрет ніколи не передається у відкритому вигляді.
- **Окремі ключі для кожного авто** та кожного телефону. Компрометація одного пристрою не зачіпає інші.
- **RTT anti-relay захист** *(Pro)* — при безключовому доступі за наближенням телефон та авто обмінюються миттєвими «пінг-понг» викликами. Це унеможливлює атаки типу «relay attack», коли зловмисник підсилює BLE-сигнал між вашим телефоном вдома та авто на парковці.
- **Локальне сховище секретів** у захищених сховищах ОС: Android Keystore та iOS Keychain.
- **Жодних серверних паролів** — усі ключі генеруються на пристроях користувача.

**Примітки:**

\* Працює лише поки головний пристрій авто має стабільний інтернет. Вбудований 4G DiLink наразі не функціонує надійно — єдиний робочий канал у Light це **Wi-Fi**. Поки авто на домашньому Wi-Fi або з підключеним USB Wi-Fi-донглом, ці функції працюють. Без активного Wi-Fi-каналу віддалений зв'язок з авто недоступний.

> [!TIP]
> Для цілодобового моніторингу у версії Light встановіть **USB Wi-Fi-донгл** в авто (наприклад, з'єднаний з портативним 4G-роутером або точкою доступу телефона). Головний пристрій залишається у мережі навіть коли авто заглушено.

У версії **Pro** модуль T-Box забезпечує:
- власний незалежний канал зв'язку (Wi-Fi-донгл не потрібен),
- окремий GPS-приймач,
- безключовий доступ із захистом від relay-атак (RTT),
- надійні push-сповіщення 24/7,
- віддалений запуск авто.
