# Налаштування

Екран налаштувань — це остання вкладка на панелі навігації. Він поділений на розділи: Автомобіль, Голосовий помічник, Зовнішній вигляд, Дані, Інформація про автомобіль та Про застосунок.

---

## Автомобіль

| Налаштування | За замовчуванням | Опис |
|---|---|---|
| Ємність батареї (Battery Capacity) | 87.4 кВт·год | Корисна ємність батареї для розрахунку енергії та оцінки запасу ходу. Змініть це значення, якщо ваш автомобіль має іншу ємність батареї. |
| Автозапуск (Auto-Start) | Увімк. | Автоматичний запуск застосунку при завантаженні системи та увімкненні екрану. |
| Резервне з'єднання (Legacy Connection) | Вимк. | Використання альтернативного (повільнішого) методу з'єднання. Увімкніть, якщо стандартний спосіб з'єднання не працює на вашій моделі головного пристрою. |
| Плаваюча панель (Quick Actions Overlay) | Вимк. | Показати або приховати плаваючу кнопку швидких дій. Дивіться [Швидкі дії](09-quick-actions.md). |

## Голосовий помічник

| Налаштування | За замовчуванням | Опис |
|---|---|---|
| Голосовий помічник (Voice Assistant) | Вимк. | Головний перемикач усіх голосових функцій. Неможливо увімкнути без API-ключа. |
| Режим активації (Activation Mode) | Фраза пробудження | **Постійне прослуховування (Always Listening)** — безперервно чекає на фразу пробудження. **Push-to-Talk** — активується лише при натисканні на плаваючу кнопку. |
| API-ключ ElevenLabs | (порожньо) | API-ключ для хмарного розпізнавання мовлення та синтезу мовлення. Відображається "Не встановлено" або "Налаштовано". Ключ надсилається виключно на сервери ElevenLabs. |
| Мова розпізнавання (Recognition Language) | Обидві | **Українська**, **Англійська** або **Обидві** (автовизначення). Вибір однієї мови покращує точність розпізнавання. |
| Фрази пробудження (Wake Phrases) | "привiт бiуайдi" / "hello byd" | Індивідуальні фрази пробудження для української та англійської мов. Знаки пунктуації видаляються автоматично. |
| Ваше ім'я (Your Name) | (порожньо) | Необов'язкове. Якщо задано, голосові підтвердження стають персоналізованими (наприклад, "Привіт, Антоне, вмикаю клімат"). |
| ID голосу TTS (TTS Voice ID) | (порожньо) | Ідентифікатор голосу ElevenLabs для індивідуального озвучення. Залиште порожнім для використання голосу за замовчуванням. |

Дивіться [Голосовий помічник](08-voice-assistant.md) для повного переліку з 26 підтримуваних команд.

## Зовнішній вигляд

| Налаштування | За замовчуванням | Опис |
|---|---|---|
| Тема (Theme) | Системна | **Системна** — слідує за темним/світлим режимом Android. **Світла** — примусово світла тема. **Темна** — примусово темна тема. Застосовується миттєво. |
| Мова (Language) | Системна | **Системна** — слідує за локаллю Android. **English** — примусово англійська. **Українська** — примусово українська. Також змінює напис на плаваючій кнопці. |

## Дані

| Дія | Опис |
|---|---|
| Очистити журнали діагностики (Clear Diagnostic Logs) | Видаляє всі записи журналу з вбудованого переглядача. Показується діалог підтвердження. |
| Експортувати діагностику (Export Diagnostic Bundle) | Зберігає комплексний діагностичний JSON-файл на пристрій. Корисно для повідомлення про проблеми. |
| Надіслати короткий звіт (Send Brief Report) | Надсилає текстовий підсумок останньої активності розробнику через Telegram. |
| Надіслати детальний звіт (Send Detailed Report) | Надсилає повний діагностичний пакет та останні журнали розробнику через Telegram. Використовуйте для розслідування конкретних проблем. |

> [!NOTE]
> Діагностичний пакет — це основний інструмент для дослідження проблем, специфічних для конкретного автомобіля. Додавайте його при повідомленні про проблему.

## Інформація про автомобіль

Цей розділ доступний лише для перегляду. Він відображає інформацію про ваш головний пристрій та автомобіль, зчитану під час з'єднання.

| Поле | Опис |
|---|---|
| Модель пристрою (Product Model) | Назва моделі головного пристрою |
| Виявлений автомобіль (Detected Vehicle) | Назва автомобіля, визначена з прошивки |
| ПЗ автомобіля (Vehicle Software) | Версія програмного забезпечення автомобіля |
| Версія MCU (MCU Version) | Версія прошивки MCU |
| Версія DiLink (DiLink Version) | Покоління DiLink та версія інтерфейсу |
| Тип DiLink (DiLink Type) | Ідентифікатор апаратного типу DiLink |
| Версія апаратури (Hardware Version) | Ревізія апаратного забезпечення |
| Ідентифікатор прошивки (Firmware ID) | Повний рядок ідентифікатора прошивки |
| Серія авто (Car Series) | Серія автомобіля BYD (наприклад, Ocean, Dynasty) |

> [!NOTE]
> Версія MCU показує "MCU_OFFLINE", коли автомобіль у стані OFF. Реальна версія з'являється після переходу автомобіля у стан ACC або READY.
