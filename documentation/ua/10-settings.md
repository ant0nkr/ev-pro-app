# Налаштування

Екран налаштувань поділений на розділи: Автомобіль, Підключення, Застосунки за замовчуванням, Голосовий помічник, Зовнішній вигляд та Діагностика й дані.

---

## Автомобіль

| Налаштування | За замовчуванням | Опис |
|---|---|---|
| Ємність батареї (Battery Capacity) | 87.4 кВт·год | Корисна ємність батареї для розрахунку енергії та оцінки запасу ходу. 21 пресет + довільне значення. |

---

## Підключення

| Налаштування | За замовчуванням | Опис |
|---|---|---|
| Автозапуск (Auto-Start) | Увімк. | Автоматичний запуск застосунку при завантаженні системи та увімкненні екрану. |
| Плаваюча панель (Quick Actions Overlay) | Вимк. | Показати або приховати плаваючу кнопку швидких дій. Дивіться [Швидкі дії](09-quick-actions.md). |
| Виводити на передній план (Bring to Foreground) | Вимк. | Застосунок автоматично виходить на передній план при увімкненні екрану автомобіля. |
| Утримувати WiFi при вимк. запаленні | Вимк. | Автоматично відновлює WiFi після вимкнення запалення. *(Розширена версія)* |
| Утримувати мобільні дані при вимк. запаленні | Вимк. | Автоматично відновлює мобільні дані після вимкнення запалення. *(Розширена версія)* |
| Утримувати Bluetooth при вимк. запаленні | Вимк. | Автоматично відновлює Bluetooth після вимкнення запалення. *(Розширена версія)* |
| Працювати на стоянці (Run While Parked) | Вимк. | Продовжувати роботу та опитування сенсорів, коли автомобіль припаркований (стан OFF). |
| Повільне опитування (Slow Poll Mode) | Вимк. | Зменшити частоту опитування для економії ресурсів. Корисно при роботі на стоянці. |
| Вимкнути монітор трафіку | Вимк. | Вимкнути сервіс BYD traffic monitor, який може заважати підключенню. |

| Налаштувати | — | Відкриває екран налаштування голосу: фрази пробудження, ваше ім'я та перемикачі окремих команд. |

## Зовнішній вигляд

| Налаштування | За замовчуванням | Опис |
|---|---|---|
| Тема (Theme) | Системна | **Системна** — слідує за темним/світлим режимом Android. **Світла** — примусово світла тема. **Темна** — примусово темна тема. |
| Мова (Language) | Системна | **Системна** — слідує за локаллю Android. **English** — примусово англійська. **Українська** — примусово українська. Також змінює мову розпізнавання голосу. |

## Діагностика й дані

| Дія | Опис |
|---|---|
| Переглянути журнали (View Logs) | Відкриває переглядач журналів з вкладками (Усі / Голос / DiLink / Авто / Опитування / Застосунок / Налагодження). |
| Очистити журнали діагностики | Видаляє всі записи журналу з вбудованого переглядача. |
| Експортувати діагностику | Зберігає комплексний діагностичний файл на пристрій. |
| Надіслати короткий звіт | Надсилає текстовий підсумок останньої активності розробнику через Telegram. |
| Надіслати детальний звіт | Надсилає повний діагностичний пакет та останні журнали розробнику через Telegram. |
| Оновити FID-конфігурацію | Примусово перезавантажити правила сумісності сенсорів. |

> [!NOTE]
> Діагностичний пакет — це основний інструмент для дослідження проблем, специфічних для конкретного автомобіля. Додавайте його при повідомленні про проблему.

| Дія | Опис |
|---|---|

## Інформація про автомобіль

Цей розділ доступний лише для перегляду. Він відображає інформацію про ваш головний пристрій та автомобіль, зчитану під час з'єднання.

| Поле | Опис |
|---|---|
| Модель пристрою | Назва моделі головного пристрою |
| Виявлений автомобіль | Назва автомобіля, визначена з прошивки |
| ПЗ автомобіля | Версія програмного забезпечення автомобіля |
| Версія MCU | Версія прошивки MCU |
| Версія DiLink | Покоління DiLink та версія інтерфейсу |
| Протокол | CAN-FD або CAN 2.0 залежно від прошивки |
| Тип привода | BEV / HEV / DM-i Plug-in Hybrid |
| Версія апаратури | Ревізія апаратного забезпечення |
| Серія авто | Серія автомобіля BYD (наприклад, Ocean, Dynasty) |

> [!NOTE]
> Версія MCU показує "MCU_OFFLINE", коли автомобіль у стані OFF. Реальна версія з'являється після переходу автомобіля у стан ACC або READY.
