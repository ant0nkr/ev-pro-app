# BYD EV Pro — Огляд

BYD EV Pro — це Android-застосунок, створений для роботи безпосередньо на головному пристрої BYD DiLink. Він забезпечує діагностику електромобіля в реальному часі, керування кліматом, голосовий асистент та програмовану автоматизацію — без жодного зовнішнього обладнання. Не потрібен OBD-адаптер, Bluetooth-донгл чи підключений смартфон. Застосунок взаємодіє безпосередньо з вбудованими системами автомобіля.

> [!NOTE]
> Для роботи застосунку необхідний доступ через ADB (Android Debug Bridge). Застосунок використовує ADB для прямого зв'язку з системами автомобіля на головному пристрої.

## Можливості

- **Панель приладів (Dashboard)** — Стан заряду батареї в реальному часі, потік потужності, прогнозований запас ходу, напруга, струм, температури двигуна, розкид напруги комірок та положення передачі.
- **Керування кліматом (Climate Control)** — Повне керування HVAC: увімкнення/вимкнення кондиціонера, цільова температура, підігрів та вентиляція сидінь (водій і пасажир), підігрів керма, обдув лобового та заднього скла.
- **Відстеження зарядних сесій (Charging)** — Автоматичне виявлення та журналювання кожної зарядки з початковим/кінцевим SOC, доданою енергією (кВт·год), тривалістю та середньою швидкістю зарядки.
- **Історія поїздок (History)** — Записи кожної поїздки: відстань, витрачена енергія, ефективність (кВт·год/100 км) та тривалість сесії.
- **Автоматизація (Automation)** — Система правил IF/THEN, що спрацьовують при зміні стану автомобіля (OFF / ACC / READY). Підтримує необов'язкове автоматичне скасування після заданого часу.
- **Голосовий асистент (Voice Assistant)** — 36 двомовних (англійська/українська) голосових команд з використанням офлайн розпізнавання VOSK, з активацією за ключовим словом або у режимі Push-to-Talk. Див. [Голосовий асистент](08-voice-assistant.md).
- **Bluetooth** — *(Функція у розробці)*
- **Плаваюча панель швидких дій (Quick Actions)** — Завжди поверх інших вікон, перетягувана кнопка, що розкривається в сітку керування кондиціонером, підігрівом керма, обдувом скла та сидіннями — доступна з будь-якого екрану. Див. [Швидкі дії](09-quick-actions.md).
- **Панель налагодження (Debug Control Panel)** — Інструмент для перевірки, що команди керування доходять до автомобіля, та зчитування фактичних значень CAN-шини. Корисно для діагностики елементів керування, що не працюють. Див. [Діагностика](11-diagnostics.md).
- **Telegram-бот** — Підключення застосунку до Telegram-бота для віддаленого отримання статусу автомобіля, деталей батареї, фото з камер та завантаження діагностичних журналів. Див. [Telegram](14-telegram.md).
- **Інтеграція з Home Assistant** — Передача телеметрії автомобіля в реальному часі до Home Assistant через webhook. HACS-компонент із 25 сенсорами та GPS-трекером. Див. [Home Assistant](15-home-assistant.md).
- **Діагностичний експорт та звіти** — Експорт повного діагностичного пакету (JSON) одним дотиком та надсилання звітів про помилки через Telegram.

## Мови

Застосунок повністю двомовний: **англійська** та **українська**. Вибір мови доступний у Налаштуваннях (Settings) і зберігається між сесіями. Голосові команди також працюють обома мовами.

---

## Чим цей застосунок НЕ є

| Цей застосунок НЕ... | Чому |
|---|---|
| Використовує OBD-II / ELM327 адаптер | Не потрібен — застосунок взаємодіє з автомобілем безпосередньо через головний пристрій |
| Використовує Bluetooth | Жодного BLE-з'єднання чи сканування |
| Працює на iOS чи десктопі | Тільки Android; розроблений для головних пристроїв BYD DiLink |
| Працює на підключеному телефоні | Застосунок працює безпосередньо на головному пристрої |
| Потребує хмарного облікового запису | Без реєстрації, без логіну, без обов'язкового інтернет-з'єднання (крім Telegram та Home Assistant) |

## Підтримувані автомобілі

Застосунок розроблений та протестований на **BYD Song Plus EV 2025** (серія Ocean, DiLink 5.0). Очікується сумісність з іншими автомобілями BYD на DiLink від 3.0 до 6.0. Повний перелік підтримуваних моделей див. у розділі [Сумісність](12-compatibility.md).

| Параметр | Значення |
|---|---|
| Основний тестовий автомобіль | BYD Song Plus EV 2025 |
| Серія | Ocean |
| Ємність батареї за замовчуванням | 87,4 кВт·год (налаштовується користувачем) |
